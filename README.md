# Tokyo Food Map / 東京美食地圖

這是一個基於 Threads 討論串「第一屆東京沒吃到會後悔大賽」開發的東京美食地圖工具。

## 功能特點
- 📍 **地理定位**：自動計算您目前位置與各餐廳的距離。
- 🍣 **分類篩選**：拉麵、燒肉、壽司、烏龍麵等多樣化分類。
- 🗺️ **地圖模式**：提供互動式地圖，視覺化呈現所有美食位置。
- ➕ **手動新增**：您可以自行擴充私房清單，資料會儲存在您的瀏覽器中。

## 開發技術
- HTML5 / CSS3 (Glassmorphism 現代設計)
- JavaScript (Vanilla JS + Leaflet.js)
- Nominatim API (反向地理編碼)
